# Staff-AI

Staff-AI is a Node.js application that provides AI-powered functionality for staff and internal tooling.

It is designed to streamline internal workflows using AI-driven automation and responses.

## Features
- AI-powered responses
- Secure API key handling via environment variables
- Modular Node.js architecture

## Tech Stack
- Node.js
- JavaScript
- OpenAI API integration
- dotenv for environment variables
